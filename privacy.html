Privacy Policy

Last updated: May 18, 2026

1. Data We Collect

We collect only the data necessary to provide the service: music files and album art uploaded by users, and TikTok authentication tokens required to publish content via the TikTok Content Posting API.

2. How We Use Data

User-uploaded content is used solely to generate video visualizers and publish them to the user's TikTok account. TikTok authentication tokens are stored securely and used exclusively for API calls on behalf of the user.

3. Third-Party Data Sharing

We do not sell, trade, or share user data with third parties. Data is shared with TikTok only as necessary to fulfill the content publishing request through the official TikTok API.

4. Data Security

We implement reasonable security measures to protect user data, including encrypted storage of authentication tokens and secure API communication.
